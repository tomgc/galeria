# Fotos

Una entrada `.md` por fotografía. El frontmatter sigue el schema de
[`src/content.config.ts`](../../content.config.ts).

No edites estos archivos a mano la primera vez — usa `npm run ingest`,
que crea el `.md` prellenado a partir del EXIF. Después rellenas
manualmente `category`, `tags`, descripciones y todo lo que falte.

> Nota: este `README.md` no es procesado como entrada de la colección
> (el loader lo excluye explícitamente).
